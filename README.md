

DaanGo is a crowdfunding/fundraiser platform.
Check it out live [here](https://DaanGoweb.vercel.app).

## Built Using

- Frontend

  - React.js
  - Cakra UI
  - TailwindCSS
  - LucidReact Icons

- Backend

  - Node.js
  - Express
  - MongoDB

- Services
  - Supabase Auth
  - Stripe
  - Uploadthing

## Features

- Email verification
- User dashboard
- Update fullname and email
- Discover fundraisers
- Search/filter
- Share fundraiser
- Details saved in draft while creating a fundraiser
- Manage fundraiser
- Post updates
- Stripe for processing donation payment
- Payment confirmation email notification
- Submit identity proofs and Bank details for review before transferring funds
